# CG-Lab

本项目为计算机图形学课程实验项目，基于 **Taichi** 框架实现一个简单的 **GPU 并行万有引力粒子群仿真系统**。程序通过 GPU 并行计算实现大量粒子的实时物理更新，并在窗口中进行可视化渲染。

## 项目结构

项目采用 `src` 布局组织代码：

```
CG-Lab
│
├─ src
│   └─ Work0
│       ├─ config.py      # 系统配置参数（粒子数量、窗口大小等）
│       ├─ physics.py     # 粒子物理系统与GPU并行计算逻辑
│       └─ main.py        # 程序入口与渲染循环
│
├─ pyproject.toml         # 项目依赖配置
├─ uv.lock                # 依赖锁定文件
└─ README.md
```

## 代码逻辑说明

项目代码采用模块化设计：

* **config.py**
  定义系统运行参数，例如粒子数量、窗口分辨率、粒子颜色等。

* **physics.py**
  使用 Taichi 定义粒子的位置与速度数据结构，并通过 GPU Kernel 实现：

  * 粒子初始化
  * 鼠标引力计算
  * 速度更新
  * 边界碰撞检测

* **main.py**
  负责程序启动与图形界面渲染：

  * 初始化 Taichi GPU 后端
  * 创建 GUI 窗口
  * 在主循环中调用物理更新函数
  * 实时绘制粒子群

## 实现功能

程序实现了一个交互式粒子系统：

* 初始化大量随机粒子
* 利用 GPU 并行计算更新粒子运动
* 鼠标位置产生引力吸引粒子
* 粒子在窗口边界发生反弹
* 实时渲染粒子群动画

## 运行方法

在项目根目录执行：

```
uv run -m src.Work0.main
```

程序启动后会打开一个窗口，移动鼠标即可观察粒子群受到引力影响产生的动态效果。
