#  计算机图形学实验二

## 📌 项目简介

本项目基于 Taichi 框架，实现了计算机图形学中的经典 **MVP（Model-View-Projection）变换管线**，并完成了从二维图形到三维几何体的逐步扩展。

通过本实验，理解了三维空间中的模型变换、视图变换、透视投影以及屏幕映射的基本原理。

---

## 🧩 实验内容

### ✅ 实验一：二维三角形旋转

* 构建一个三角形（3 个顶点）
* 实现绕 Z 轴的模型旋转
* 引入视图变换（相机位置）
* 使用透视投影矩阵进行空间映射
* 将三维坐标转换到屏幕坐标并绘制

## 🧠 核心原理

本项目主要涉及以下图形学基础：

* **Model Transform（模型变换）**：实现物体的旋转
* **View Transform（视图变换）**：模拟相机位置变化
* **Projection Transform（投影变换）**：实现三维到二维的透视映射
* **齐次坐标与透视除法**：完成空间坐标归一化
* **Viewport Transform（视口变换）**：映射到屏幕坐标

整体流程如下：

3D 顶点 → MVP 变换 → 裁剪空间 → NDC → 屏幕坐标
