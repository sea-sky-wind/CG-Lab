# CG-Lab: 曲线绘制实验

本仓库包含计算机图形学实验相关的曲线绘制程序，包括基础贝塞尔曲线、反走样优化，以及B样条曲线实现。


---

## 实验文件说明

### 1. 必做：基础贝塞尔曲线
- 功能：
  - 鼠标点击添加控制点
  - 自动绘制红色控制点、灰色控制多边形、绿色贝塞尔曲线
  - 键盘 `C` 清空画布

- 运行示例：
![alt text](必做.gif)

---

### 2. 选做：反走样贝塞尔曲线
- 功能：
  - 在基础贝塞尔曲线上增加亚像素平滑渲染
  - 曲线边缘视觉平滑，无明显阶梯状锯齿
- 运行示例：
![alt text](选做1.gif)

---

### 3. 选做：B样条曲线
- 功能：
  - 支持三次均匀B样条曲线绘制
  - 键盘 `B` 切换贝塞尔/ B样条模式
  - 局部控制特性：修改单个控制点只影响局部曲线段
- 运行示例：
![alt text](选做2.gif)
